 Healthcare Cost Prediction using Machine Learning
1. Project Title
Healthcare Cost Prediction using Machine Learning
2. Project Description
This project focuses on predicting healthcare costs using various Machine Learning regression algorithms. The prediction is based on patient-related attributes such as age, medical condition, admission type, and hospital stay duration.
The project implements and compares multiple models including Linear Regression, Decision Tree, Random Forest, Support Vector Regression (SVR), Gradient Boosting, and K-Nearest Neighbours (KNN).
The objective is to identify the most accurate and reliable model for healthcare cost prediction.
3. Objectives
•	To apply Machine Learning techniques for healthcare cost prediction 
•	To compare different regression models 
•	To improve prediction accuracy using preprocessing techniques 
•	To evaluate model performance using standard metrics
